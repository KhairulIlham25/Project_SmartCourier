# SMART COURIER PROJECT

Mata Kuliah: PAA
Dosen Pengampu: Tekad Matulatan, S.T., M.Tech

>Tugas Project
Membuat program smart courier dimana sang kurir berusaha mengantarkan paket dari satu titik (source) ke titik lainnya (destination).
Pada proyek ini pengguna bisa mengupload map kustom yang bisa dibaca oleh kode program, kemudian ikon kurir akan bergerak dari tempat awal hingga mencapai destinasi pada bagian map yang dibaca sebagai jalan, dan menghindari bagian lain yang bukan termasuk jalan.
